Message to Author Wordpress plugin allows visitors to message to the author of the post or page or any custom posts. Which can be useful to give feedback about a particular post to an author of the post directly from the position. It works fine with any custom posts; thus,  you can use it as an inquiry form for WooCommerce product, job posting, etc.


== Description ==
This plugin allows you to message directly to the author of the post, page or any other custom post. You can use it as inquiry form, feedback form or something to share with the author/admin of the post,  Author may check messages right from the admin panel and also there is a setting to send an email to the author regarding the new inquiry

The plugin has two front-end layout:
1. Message box
2. popup


You can help contribute your code here: https://github.com/pathusutariya/message-to-author

------USER INSTRUCTION-----

1. You can use shortcode for display at your location on the page `[message2author]` `[message2author style="popup"]`  or `[message2author style="messagebox"]` for different styles
2. active after every content from admin settings page


== Installation ==
From your WordPress dashboard:

- Visit \Plugins > Add New\
- Search for \Message to Author\
- Activate Message to author from your Plugins page.

From WordPress.org

- Download Message to Author.
- Upload the `\message-to-author\ ` directory to your `/wp-content/plugins/ ` directory, using your favorite method (ftp, sftp, scp, etc...)
- Activate message-to-author from your Plugins page.
